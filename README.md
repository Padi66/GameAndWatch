Présentation de la demo Game & Watch de Alexis ROGER.


Dans le jeu, vous incarnez un hacker qui cherche à s'infiltrer dans un ordinateur. Pour cela, il devra résoudre trois mini-jeux afin de franchir les trois couches qui protègent le noyau de stockage des données. Pour y parvenir, il devra obtenir les codes d'accès aux différentes couches, cachés dans les différents mini-jeux.

Le premier mini-jeu est inspiré des Game & Watch classiques, avec une touche de modernité pour coller au thème du jeu. On y retrouve l’avatar informatique de notre hacker, qui doit rassembler un maximum de données bleues afin d’obtenir le prochain code qui déverrouillera la deuxième couche. Malheureusement, le mini-jeu est bien protégé et le hacker devra éviter de télécharger les données rouges contenant des malwares, qui l’éjecteront automatiquement de la simulation.

Le gameplay rappelle fortement les Game & Watch classiques, avec des slots de déplacement prédéfinis dans lesquels le joueur peut se déplacer afin de récupérer ou esquiver les données. Les données tombent du haut vers le bas de l’écran de manière linéaire. Chaque donnée bleue collectée augmente le score du joueur de 10 points, tandis que la collecte d’une donnée rouge enlève une vie. Le joueur commence avec 3 vies.

Une fois les 200 points atteints, une fenêtre s’ouvre avec le code permettant de passer au niveau suivant. Le joueur peut alors choisir de continuer le mini-jeu ou de s’arrêter et retourner au menu pour attaquer un autre mini-jeu.

Au fur et à mesure que la partie progresse, la difficulté augmente : la vitesse de déplacement des données s’accélère et leur nombre à l’écran augmente également. Le joueur devra donc faire preuve de réflexes et d’habileté pour collecter un maximum de points. Le jeu est infini tant que le hacker ne se fait pas déconnecter de la simulation à cause des données rouges.

La musique et le visuel apportent une touche plus moderne tout en conservant l’inspiration Game & Watch. La bande-son s’inspire du style 8-bit de l’époque, mais avec une approche plus électronique et entraînante. Elle accompagne parfaitement le gameplay et maintient le joueur sous tension. Le visuel, quant à lui, mélange tilemaps et images fixes afin de garantir une cohérence esthétique avec la narration du jeu.

Le menu principal présente les trois couches que le joueur devra traverser pour hacker le système. Lorsqu’il clique sur une des couches, une fenêtre s’ouvre et lui demande d’entrer le mot de passe. Pour les phases de test, le code de la première couche (la plus extérieure) est déjà inscrit sur un post-it laissé par le hacker avant l’arrivée du joueur : 1234.

Concernant la narration (provisoire), le joueur pourra choisir d’incarner un hacker « chevalier blanc » ou « chevalier noir ». Son objectif sera donc différent selon son choix, mais la manière d’y parvenir restera la même : résoudre les différents mini-jeux afin de pirater les couches qui protègent le noyau central contenant les données que le hacker souhaite récupérer.
